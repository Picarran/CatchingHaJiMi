# CATCHING HAJIMI

项目名称：逮捕哈基米

# 数据库

- 请在mysql内新建一个数据库```catchinghajimi```，建议在账户名:```root``` 密码:```123456``` 的账户下创建。否则请自行修改```MyCatHome\src\main\resources\application.yml```内数据库的```username```和```password```

- 数据库表设计见```MyCatHome\schema.sql```

# 项目设置

- IDEA中环境配置，确定Settings和Project Structure中JDK对应的版本均为Java 17。
- 其他配置详见```MyCatHome\pom.xml```

# 项目运行方法:

- 后端运行/jar包运行后，浏览器访问```http://localhost:8080/user/login```

# 玩法介绍：

共设置三个地图。游戏开始后，点击空格放置障碍，阻止哈基米到达地图边界逃跑。注意：哈基米只能沿“上/下/左/右”四个方向移动。

提示：尽量封住最短路径，优先包围中心附近的空位。

玩法可见 ```视频演示.mp4```

# 项目功能概要：
- 玩家注册登录
- 基于dfs规划哈基米逃跑路线
- 游戏最佳记录设置
- 游戏玩家排行榜

# 游戏创意
- 灵感来源于经典迷宫追捕类游戏，规则简单但考验玩家策略：玩家通过放障碍封堵哈基米的逃跑路径，并设置玩家最好成绩排名。

- 三张地图布局不同，以供玩家不同策略应对。后续地图可拓展。

- 哈基米按上下左右移动，逃跑路线由 DFS 计算。

- 后续扩展：可以修改哈基米的移动次数、限时模式或特殊障碍等。
